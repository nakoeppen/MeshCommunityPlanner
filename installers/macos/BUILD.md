# macOS Build Guide (Clean Image)

Assumes a fresh macOS install (Big Sur 11.0 or later) with nothing installed.

## 1. Install Homebrew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Follow the post-install instructions to add `brew` to your PATH.

## 2. Install system tools

```bash
brew install python@3.12 node git dos2unix create-dmg gh
```

> **Important:** Use Python 3.12. Python 3.14 is too new — pydantic-core does not support it yet.

## 3. Authenticate GitHub CLI

```bash
gh auth login
```

## 4. Clone the repo

```bash
git clone https://github.com/PapaSierra555/MeshCommunityPlanner.git
cd MeshCommunityPlanner
```

## 5. Fix line endings on shell scripts

```bash
dos2unix installers/macos/build_dmg.sh
chmod +x installers/macos/build_dmg.sh
```

## 6. Python environment + dependencies

```bash
python3.12 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pip install pyinstaller
```

## 7. Build frontend

```bash
cd frontend && npm install && npx vite build && cd ..
```

## 8. Build PyInstaller bundle

```bash
python3 -m PyInstaller installers/mesh_planner.spec --noconfirm
```

This produces:
- `dist/MeshCommunityPlanner/` — standalone directory (run from terminal)
- `dist/Mesh Community Planner.app/` — proper .app bundle

## 9. Build DMG

```bash
./installers/macos/build_dmg.sh
```

Output: `installers/dist/MeshCommunityPlanner-1.2.0.dmg`

## 10. Test the DMG

```bash
hdiutil attach installers/dist/MeshCommunityPlanner-1.2.0.dmg
cp -R "/Volumes/Mesh Community Planner/MeshCommunityPlanner.app" /Applications/
xattr -cr /Applications/MeshCommunityPlanner.app
open -n /Applications/MeshCommunityPlanner.app
```

Should start the server and open your browser to `http://127.0.0.1:8321`.

Eject the DMG from Finder when done testing.

## 11. Upload to release

```bash
gh release upload v1.2.0 installers/dist/MeshCommunityPlanner-1.2.0.dmg
```

## What the DMG Contains

```
MeshCommunityPlanner.app/
  Contents/
    Info.plist                    # App metadata (generated by PyInstaller)
    MacOS/
      MeshCommunityPlanner        # Launcher script (starts server + opens browser)
      MeshCommunityPlanner-bin    # Actual PyInstaller binary
    Frameworks/
      Python3                     # Python shared library
    Resources/
      AppIcon.icns                # App icon
      base_library.zip            # Python standard library
```

## Gatekeeper (unsigned app)

On first launch, macOS blocks unsigned apps:
1. Right-click app > **Open** > **Open** in dialog
2. Or: `xattr -cr /Applications/MeshCommunityPlanner.app`

## Logs

Server output is logged to `~/Library/Logs/MeshCommunityPlanner.log`

## Run (command line, no .app)

```bash
./dist/MeshCommunityPlanner/MeshCommunityPlanner &
open http://127.0.0.1:8321
```

## Optional: Code Signing

```bash
codesign --force --deep --sign - "dist/Mesh Community Planner.app"
```
