# macOS Build Guide

## Prerequisites

```bash
brew install python@3.9 node   # or python@3.11, python@3.13
pip3 install pyinstaller
```

- macOS 11.0+ (Big Sur or later)
- Python 3.9+ (tested with 3.9.6 and 3.13)
- Node.js 18+

## Build

```bash
git clone https://github.com/PapaSierra555/MeshCommunityPlanner.git
cd MeshCommunityPlanner

pip3 install -r requirements.txt

cd frontend && npm install && npx vite build && cd ..

python3 -m PyInstaller installers/mesh_planner.spec --noconfirm
```

PyInstaller produces two outputs on macOS:
- `dist/MeshCommunityPlanner/` — standalone directory (run from terminal)
- `dist/Mesh Community Planner.app/` — proper .app bundle

## Run (command line, no .app)

```bash
./dist/MeshCommunityPlanner/MeshCommunityPlanner &
open http://127.0.0.1:8321
```

## Build DMG

```bash
chmod +x installers/macos/build_dmg.sh
./installers/macos/build_dmg.sh
```

Output: `installers/dist/MeshCommunityPlanner-1.0.0.dmg`

The DMG script:
1. Takes the PyInstaller `.app` bundle from `dist/`
2. Generates `AppIcon.icns` from branding PNGs
3. Adds a launcher script (starts server + opens browser automatically)
4. Wraps everything into a DMG with an Applications shortcut

## What the DMG Contains

```
Mesh Community Planner.app/
  Contents/
    Info.plist                    # App metadata (generated by PyInstaller)
    MacOS/
      MeshCommunityPlanner        # Launcher script (starts server + opens browser)
      MeshCommunityPlanner-bin    # Actual PyInstaller binary
    Frameworks/
      Python3                     # Python shared library
    Resources/
      AppIcon.icns                # App icon
      base_library.zip            # Python standard library
```

## Gatekeeper (unsigned app)

On first launch, macOS blocks unsigned apps:
1. Right-click app > **Open** > **Open** in dialog
2. Or: `xattr -cr "/Applications/Mesh Community Planner.app"`

## Logs

Server output is logged to `~/Library/Logs/MeshCommunityPlanner.log`

## Optional: Code Signing

```bash
codesign --force --deep --sign - "dist/Mesh Community Planner.app"
```
